
# Backend Life After UB

Backend Life After UB dibuat menggunakan Laravel sebagai REST API untuk mendukung sistem rekomendasi karir mahasiswa dan alumni Universitas Brawijaya. Backend ini digunakan untuk mengelola autentikasi pengguna, pengolahan database, manajemen data admin, serta integrasi seluruh fitur website secara terstruktur dan terhubung dengan frontend.

## Fitur
- Authentication Login & Register
- REST API Dashboard User & Admin
- CRUD User, Karir, Alumni, dan Rekomendasi
- Monitoring Aktivitas User

## Tools & Environment
- Laravel
- MySQL
- XAMPP
- Laragon
- Visual Studio Code
- Sanctum Authentication
